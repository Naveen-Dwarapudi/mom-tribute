# 💙 For Annapurna Dwarapudi — A Tribute Website

> *A son's greatest gift to his queen.*

Built with love by **Naveen Dwarapudi** — a heartfelt, feature-rich tribute website dedicated to his mother, **Annapurna Dwarapudi**.

---

## ✨ Features

- **Cinematic Loading Screen** — A gentle crown animation while the page loads
- **Custom Cursor** — Soft blue cursor with a follower effect
- **Animated Hero Section** — Floating crown above a circular photo frame with a spinning golden ring
- **Live Photo Upload** — Click any photo frame to upload her real photos (no server needed)
- **Lightbox Gallery** — Click any uploaded photo to view it in full screen
- **Heartfelt Message** — Personal letter from Naveen to Annapurna
- **Superpower Cards** — 6 beautifully animated quality cards
- **Journey Timeline** — A visual story of her life and sacrifices
- **Floating Particles** — Soft sparkles and emoji petals drifting across the screen
- **Scroll Reveal Animations** — Every section fades in gracefully as you scroll
- **Parallax Glow Effect** — The hero glow follows the mouse cursor
- **Typewriter Effect** — The dedication title types itself when scrolled into view
- **Fully Responsive** — Works beautifully on mobile, tablet & desktop
- **Copyright Footer** — Properly attributed to Naveen Dwarapudi

## 🚀 How to Host on GitHub Pages

1. Create a new repository on GitHub (e.g., `annapurna-tribute`)
2. Upload all files keeping the folder structure intact
3. Go to **Settings → Pages**
4. Under **Source**, select `main` branch and `/ (root)` folder
5. Click **Save** — your site will be live at:
   `https://yourusername.github.io/annapurna-tribute`

## 🖼️ Adding Photos

- Click the **circular photo frame** in the hero to add her main portrait
- Click any of the **5 photo slots** in the gallery to add memories
- Photos are loaded locally — click an uploaded photo to view it in the lightbox
